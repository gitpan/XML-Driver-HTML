Copyleft, der Spitzname für die GNU General Public License.

Unter diesem Label ist seit den späten 80ern eine Bewegung entstanden, die alles was vorher in der Softwareindustrie für möglich gehalten wurde, auf den Kopf gestellt hat.

Diese Site ermöglicht Authoren freier Software ein gemeinsames Arbeiten und Veröffentlichen im Netz.

Wer ein Project unter der GPL oder einer ähnlich freie License hat und noch einen schönen Platz im Web sucht, kann mir Michael Köhne eine eMail schreiben. Hierin sollte dann kurz das Project beschrieben werden, und mir die realnamen, pseudos und passwörter mitgeteilt werden, damit ich das Project einrichten kann.
