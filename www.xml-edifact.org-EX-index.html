Example Files

Most of the examples will raise an error with validating XML parsers, as those messages are real life and real life UN/EDIFACT always needs code list extensions.

Now unknown codes, are translated using the unknown:code attribute. This will break any validating parser, but would still allow processing by non validating parsers, who might want to fix the codelist.

A last warning. There is no CSS or XSL attached to the XML files. The 0.35 XML and EDI files are not intedend to become displayed in a browser, but processed in a stream.
